# Shopping Cart - DOM Workshop

A simple, interactive shopping cart application built using **HTML5**, **Bootstrap 5**, and **Vanilla JavaScript**. This project demonstrates basic DOM manipulation techniques, event handling, and dynamic UI updating.

## 🚀 Features

* **Adjust Quantities:** Increase or decrease product quantities using the plus and minus buttons.
* **Dynamic Total:** The total price updates automatically as you change quantities or delete items.
* **Remove Items:** Delete products from the cart entirely using the trash icon.
* **Interactive UI:** "Like" products by toggling the heart icon color.
* **Responsive Design:** Built with Bootstrap to ensure it looks great on mobile and desktop.

## 🛠️ Technologies Used

* **HTML5:** Structure of the application.
* **CSS3 / Bootstrap 5:** Styling and layout.
* **Font Awesome:** Icons for the interactive elements.
* **JavaScript (ES6+):** Logic for DOM manipulation and price calculations.
